# DIRECTV The Marketing ZONE

A React landing page for DIRECTV's Marketing Zone, featuring a featured content carousel and support section.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

The built files will be in the `dist` folder.

## Project Structure

- `src/components/` - Reusable React components (Header, FeaturedCarousel, SupportSection, Footer)
- `src/App.jsx` - Main application layout
- `public/` - Static assets
